# Gox

## Overview

Gox provides essential utilities and data structures that extend Go's standard library.  
Built with zero dependencies.

## Roadmap

Planned packages and features:

- [ ] **slices** (Map, Filter, Reduce, etc...)
- [ ] **maps** (Enhanced map operations and utilities)
- [ ] **concurrent** (Worker pools, rate limiters, circuit breakers)
- [ ] **validation** (Struct and value validation)
- [ ] **cache** (LRU/LFU cache implementations)
